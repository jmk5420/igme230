Intro

I didn’t use any templates for this project. All of the images and code was all my own and made by me. The big difficulties for this assignment was making a grid. I had numerous problems keeping track of the code. These problems include keeping the rows and columns to align with the content as well as making the mobile version to work with the grid. In the end, it did make it easier and quicker to make the website and align the content

Goals/Audience

Employers

Clean an enticing for ease of use. Visually popping and memorable. This is so that my site stays with an employer’s mind so that they will come back to me. I also need to have contact information so that they can find me.

People looking for freelance work

I want to show off! I want to demonstrate a lot of different skills that I have so that I can have a wide range of requests for project. EX: animation, UI, UX, graphic design, photography.

People who are coming from my 3D blog on Instagram

These people have most likely followed the link to my website on my Instagram. What I want from them is to get interested in following my work on my website as well sharing my work. I also want to consider that most of these people are using mobile.

Design Decisions

All of the images that I used for this project are pieces of the process that I made during the projects that I have done in the past. For color I used the red that I usually use for my own branding because it a good attention getting color. For my font I used Monserrat because it a web safe font with a lot of weights.

Above and Beyond

For my above and beyond instead of using the plugin that I originally planned on using I used the JavaScript library of modernizr. What this plugin does is makes it so that certain aspects of the website will run better on other web browsers. This is meant to help with flex-box and iframes as they can sometimes have issues running on other browsers.
